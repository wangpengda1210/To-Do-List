Percentage

Write a function get_percentage that takes a real number and represents it as a percentage. Also, add an optional parameter round_digits, which indicates precision (the number of digits after the point). If it is not specified, round the result to the nearest integer. Here are some examples (the commented lines represent the intended output of your program; the whitespace after # should not be a part of your output):

print(get_percentage(0.0123))      # 1%
print(get_percentage(0.0123, 0))   # 1.0%
print(get_percentage(0.0123, 1))   # 1.2%
print(get_percentage(0.0123, 10))  # 1.23%
print(get_percentage(0.0296, 1))   # 3.0%

Don't forget to add the percent sign % after the number. Your function should return the result, not print it. The returned percentage should be a string.

Please, use a built-in function round() to get the specified precision. The function takes up to two arguments, number and ndigits.

Just implement your function, you don't have to handle input or call it in your code.
